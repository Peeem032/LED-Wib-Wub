# 💡 LED Wib-Wub
**LED Wib-Wub** is a fun, interactive hardware project built using Arduino. The name "Wib-Wub" (meaning sparkling, blinking, or flashing in Thai) perfectly describes its purpose: creating dynamic, eye-catching light patterns and sequences using an array of 10 LEDs. 

## Key Technical Components & Features
*   **Microcontroller Programming (`10-LED.ino`):** The core logic is written in C++ for the Arduino platform, utilizing loops, pin targeting, and precise timing functions to control the light sequence.
*   **Hardware Integration:** Designed to interface directly with physical electronic components, managing the GPIO (General Purpose Input/Output) pins to drive 10 individual LEDs safely and effectively.
*   **Pattern Generation:** Demonstrates the fundamentals of embedded systems by creating custom "wib-wub" flashing visual effects, which can be adapted for displays, indicators, or decorative lighting.
*   **Open Source:** The project includes a standard `LICENSE`, making the circuitry code freely available for other makers and hobbyists to learn from or modify.
